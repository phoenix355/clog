# Nginx安装

1、下载源码包

wget https://nginx.org/download/nginx-1.20.2.tar.gz -P /opt/software/server
1

2、解压

cd /opt/software/server
tar -zvxf nginx-1.20.2.tar.gz
1
2

3、检查平台安装环境

--prefix=/usr/local/nginx 是 nginx 编译安装的目录（推荐），安装完后会在此目录下生成相关文件，相关配置文件也在此目录

cd nginx-1.20.2
./configure --prefix=/usr/local/nginx
1
2

如果前面的依赖库都安装成功后，执行 ./configure --prefix=/usr/local/nginx 命令会显示一些环境信息。如果出现错误，一般是依赖库没有安装完成，可按照错误提示信息进行所缺的依赖库安装。

# Ngnix 常用命令

  • 启动服务
nginx

  • 重新加载服务
nginx -s reload

  • 停止服务
nginx -s stop
1
  • 查看服务进程
ps -ef | grep nginx
1
  • 检查Nginx配置是否正确
nginx -t

# Nginx配置

打开/nginx/conf、nginx.conf配置文件

server {
        listen       80;    //映射出的端口
        server_name  abc.com; //访问的域名或ip(默认是localhost，有时会被占用，是不是随便输个域名都可以？待验证)
        location / {
	    root   /usr/local/nginx/html/prod_dist; //放代码的路径（index.html的根目录）
            index  index.html index.htm;
        }
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
}
